# ios-ai-delivery-workflow

Notes, templates, and experiments for AI-assisted project delivery in real mobile projects.

## What This Repo Is For

This repository is used to organize one practical topic:

how AI can help experienced engineers deliver real projects more clearly, more safely, and with better reuse.

The focus is not on hype, benchmark screenshots, or generic prompt lists.

The focus is on:

- planning before coding
- defining risk boundaries
- low-risk iteration in legacy projects
- better handoff and retrospectives
- reusable workflow assets

## Structure

- `articles/`
  article index and links to published writing

- `templates/`
  reusable templates for planning, handoff, collaboration memory, and risk boundaries

- `notes/`
  desensitized workflow notes and public example cases

- `tools/`
  reserved for future lightweight workflow experiments

## Current Topics
